Form Widget
 Previous  Next
You can drag the form widget from the widget box in to the site pages. This widget will create a custom form.

Here you can add form fields. The form field types are
  • Text Box
  • Text Area
  • Select Box
  • Check Box
  • Radio Button
Once all the form fields are added, Click on the update button, it will add the forms into the site page. Once the form added you can edit the form at any time. .
