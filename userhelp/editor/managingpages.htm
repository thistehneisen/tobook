The editor will load the template and also the default pages of the template. The default pages means the template it self may contains the pags like about us, contact us , services etc.You can modify the page layout and contents.

Also you can create your own pages.

1. Page Listing

   You can see the pages created by the editor from the header area. While choosing the page, the editor will load the corresponding page and its details.

     
     



2. Add New Page
     
     By clicking on the 'Add New Page' you can create a new page.

     

A : Page Title

The page title is the name of the page. You can add a text with space, as the entered text will be the name of the page and the editor will create a url from that name.

B : Page Type

There are two types of pages. The subpage is a normal subpage. You can add or edit text contents in that page. Another one is 'Guest Book'. This type of page has a built in PHP code for comment adding.

 

3.Page Listing
This section lists all the pages in the creating site. You can add/edit/delete the pages in the site.
