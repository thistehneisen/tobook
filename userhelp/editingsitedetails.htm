Editing Site Details
 Previous  Next
With the Simple Build Type sites, editing the site details is little bit different from the Advanced Build Type sites. This is because, with Simple Build Type templates, iScripts EasyCreate 3.0 has taken special care against the basic site layout being tampered by novice users.


To edit the details of a site, go to Site Manager. Click on 'Edit' link displayed against the site you want to edit. Depending on the build type of the site selected, you are redirected to two pages:


You can edit the following details in this section:
  • Logo
  • Company Name
  • Site Color
  • Site Caption
  • Page title
  • Contact Email
  • Meta Keywords
  • Meta Description
