Managing Sites
 Previous  Next
As the site administrator, you can preview or delete the sites created by the users. To manage the sites created by the users, go to the 'Site Manager' from the main menu. All the sites created by all the users are displayed. Click on 'Preview' button provided against the site which you require to preview. The site opens in a new page in preview mode.

To delete a site, click on the 'Delete' link against the site which you want to delete. On confirmation, the site is deleted and all the files created will be removed from the web server.
