Managing Users
 Previous  Next
The users registered with iScripts EasyCreate 3.0 can be viewed and managed from this section.

To edit the profile of a user, click on the edit link against the user. In the ensuing screen, edit the details and click 'Save'.

To delete a user, click on the delete link against the user. Confirming your delete operation, the user and all the sites created by the user will be deleted. As a word of caution, please make sure that you are deleting the one you have intended since this action is irreversible and once deleted, the user or the sites created by the user cannot be reproduced again.
