System Requirements
 Previous  Next
To run the iScripts EasyCreate 3.0, the following are required:

Server Side
  • A Web Server (Apache/IIS or any other) capable of running PHP and MySQL.
  • PHP 4.x.x or above with GD Support (GD Library is an Image Manipulation library)
  • MySQL Database v3.23.58 or above.


Client Side
  • Operating System: Windows 95, 98, ME, NT, 2000 or XP
  • Web Browser: Internet Explorer v5.5 or above
  • JavaScript: Must be enabled
