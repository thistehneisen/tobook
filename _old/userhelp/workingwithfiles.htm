Working with Files
 Previous  
Once you have published a site and uploaded your files, if you have any more files to upload to your published site, you can use the 'File Manager' to do the same.

The File Manager can be used to upload any extra files which are not provided by the site. Examples for these kind of files are catalogues, brochures, documents or any other downloadable files. You can upload any kind of files using this utility. While uploading files make sure that you are not overwriting any required files with unintended ones.

To manage your files online, click on File Manager. All the files you have uploaded through the File Manager are displayed in this screen. You can either check the status of the files uploaded or delete the files.

1. Checking Status

Click on the 'Check Status' link displayed against the image you want to check status. You are now asked to enter the FTP details of the server. After entering the required information, click on 'Check Status'. If the FTP details you have provided are verified correct by the web server, your file's details are displayed.

2. Deleting Files

Click on the 'Delete' link displayed against the image you want to delete. You are now asked to enter the FTP details of the server. After entering the required information, click on 'Delete'. If the FTP details you have provided are verified correct by the web server, your file is deleted.

3. Uploading files

Uploading files to FTP server also can be done with the File Manager. Click on 'Upload a new file' link in the bottom part of the 'File Manager'. Select the file you want to upload, select the site, enter its FTP details and click on 'Upload'. Your file is uploaded to the server if the FTP details you have provided is correct and the status is displayed.
