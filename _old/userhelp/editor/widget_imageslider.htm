Image Slider Widget
 Previous  Next
You can drag the slider widget from the widget box in to the site pages. This widget will create an image slider.

In the sildeshow widget, you the following options
  • Slide show - View the images in the slideshow
  • View Gallery - View the images in the site gallery.
  • My Uploads - View your previous uploaded images
  • Upload Image - Upload a new image
  • Settings - Change the settings of the image slider
