Editing Site Details
 Previous  Next
While creating the site you can set the site detials like site name, company name , logo etc. Once it set, you can edit the details at any time.

1.To edit the site details use the 'Back' button in the footer section of the editor or click 'Edit site details' lik. It will redirect you to a page where you can edit the site details.

You can edit the following details in this section:
  • Logo
  • Site Name
  • Website Title
  • Site Color
  • Site Description
  • Google Analytics Code
  • Meta Title
  • Meta Keywords
  • Meta Description
