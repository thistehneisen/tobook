What is Easycreate editor ?
Previous  Next

The editor provides an option to change the structure and style of your site. The editor contains a webpage as many panels. You can change the positions of the panels by dragging the panels. Also you can edit the html contents of the panels. The editor by default contains some image slider, menus, social share icons etc. By clicking the items, you can edit the parameters and settings of the items.

The default widgets in the templates are,

Images

Image slider

Menu

Social share icons
